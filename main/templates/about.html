About

The College API is a RESTful API that students can interact with in the initial phases of their college search when making an initial college list draft. Students may use the API for searching for and filtering colleges based on basic information such as city, state, college type (private/public), and other factors.
The information available is presented in a JSON (JavaScript Object Notation) format, consisting of attribute-value pairs.
Available API Endpoints:
  • Colleges
  • City
  • State
  • Type
  • Calendar
  • Setting
