Documentation

Welcome to the College API Documentation page. This site explains how to access college data.

Authentication fingerprint

Authentication is not needed to explore the College API. This is an open API.

Requests

Root HTTP Request: https://www.collegesapi/herokuapp.com/api/v1


Colleges
GET /colleges/all

Returns all colleges.

Name Type Description
college-name string Name of college
type string Type of college (private, public, ...)
city string Location of college (city)
state string Location of college (state)
academic-calendar string Term system run by college
campus-setting string Nature of college surroundings

Example Query
https://collegesapi.herokuapp.com/api/v1/colleges/all

Getting a specific college

You can access specific colleges by following its endpoint:

GET /colleges/{college-name}

Filtering Colleges

You can get collges that have specific properties by these parameter inputs:

GET /colleges/parameter/{value}
Parameter Type Description
type string Colleges with the specified type are returned
city string Colleges with the specified city are returned
state string Colleges with the specified state are returned
calendar string Colleges with the specified calendar are returned
setting string Colleges with the specified setting are returned
